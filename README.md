# ESPlay
Have you ever wanted to reuse your ESP32, Buttons and mini 0.96 inch OLED display ? No ?

Well now you can !

Flash this firmware in Arduino IDE
And Enjoy your "gaming device"

Connect :

  Display :

    SCL - D22
    SDA - D21

  Buttons :

    UP Left Button - D15
    Down Left Button - D14
    UP Right Button - D13
    UP Down Button - D12
